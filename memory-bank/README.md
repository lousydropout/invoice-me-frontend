# Memory Bank

This directory contains structured knowledge about the InvoiceMe project for AI assistants and developers.

## Structure

- `project-overview.md` - High-level project description and architecture
- `frontend-prd.md` - Frontend Product Requirements Document
- `api-reference.md` - Backend API endpoints and OpenAPI spec details
- `deployment.md` - Deployment configuration and infrastructure details
- `development-notes.md` - Development patterns, conventions, and gotchas
- `task-list.md` - Frontend development task checklist

## Purpose

The memory bank serves as a persistent knowledge base that helps maintain context across sessions and provides quick reference for project details.
